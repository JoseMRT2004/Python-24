## üß© Desaf√≠o L√≥gico: Problema de Monty Hall

**Problema:** En el juego de Monty Hall, hay tres puertas. Detr√°s de una puerta hay un coche (el premio) y detr√°s de las otras dos hay cabras. El jugador elige una puerta, luego el presentador abre una de las otras dos puertas, revelando una cabra. El jugador tiene la opci√≥n de quedarse con su elecci√≥n original o cambiar a la otra puerta.

### C√≥digo Base

```python
import random

def elegir_puerta():
    """
    Permite al jugador elegir una puerta al azar.
    Devuelve el n√∫mero de la puerta elegida (1, 2 o 3).
    """
    return random.randint(1, 3)

def abrir_puerta(puerta_elegida, puertas):
    """
    El presentador abre una puerta que no es la elegida por el jugador
    y que tiene una cabra detr√°s.
    Devuelve el n√∫mero de la puerta abierta.
    """
    # Encuentra las puertas que no han sido elegidas y que tienen cabras
    puertas_abiertas = [puerta for puerta in puertas if puerta != puerta_elegida and puertas[puerta] == 'cabra']
    return random.choice(puertas_abiertas)

def simular_juego(cambiar: bool) -> bool:
    """
    Simula un juego de Monty Hall.
    Si 'cambiar' es True, el jugador cambia su elecci√≥n.
    Devuelve True si el jugador gana el coche, False si no.
    """
    # Inicializa las puertas
    puertas = {1: 'cabra', 2: 'cabra', 3: 'coche'}
    # Mezcla las puertas
    random.shuffle(list(puertas.values()))

    # El jugador elige una puerta
    puerta_elegida = elegir_puerta()
    
    # El presentador abre una puerta
    puerta_abierta = abrir_puerta(puerta_elegida, puertas)

    # Aqu√≠ es donde debes completar la l√≥gica:
    # Si el jugador decide cambiar, determina cu√°l es la otra puerta
    # y verifica si el jugador gana el coche.

    # ¬°Tu implementaci√≥n va aqu√≠!
    if cambiar:
        # Encuentra la puerta que no fue elegida ni abierta
        puerta_cambiada = 6 - puerta_elegida - puerta_abierta  # Suma de 1+2+3 es 6
        return puertas[puerta_cambiada] == 'coche'
    else:
        return puertas[puerta_elegida] == 'coche'

def simular_varios_juegos(num_juegos: int, cambiar: bool) -> int:
    """
    Simula m√∫ltiples juegos y cuenta cu√°ntas veces el jugador gana.
    """
    ganados = 0
    for _ in range(num_juegos):
        if simular_juego(cambiar):
            ganados += 1
    return ganados

# Ejemplo de uso
num_juegos = 1000
ganados_cambiar = simular_varios_juegos(num_juegos, cambiar=True)
ganados_no_cambiar = simular_varios_juegos(num_juegos, cambiar=False)

print(f"Ganas al cambiar: {ganados_cambiar} de {num_juegos} juegos.")
print(f"Ganas al no cambiar: {ganados_no_cambiar} de {num_juegos} juegos.")```
